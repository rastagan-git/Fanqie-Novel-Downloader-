# Fanqie-Novel-Downloader-
这是一个基于 Python 的番茄小说下载工具，支持自动抓取整本书籍内容、处理字体解密映射，并合并为单个 TXT 文件。
## 环境搭建教程

如果你是第一次使用，请按照以下步骤配置环境：

### 1. 创建虚拟环境
在项目根目录下打开 CMD 或 PowerShell，输入以下命令：
```bash
python -m venv venv
```

### 2.激活虚拟环境并下载依赖
```
call venv\Scripts\activate
pip install -r requirements.txt
```

## 使用教程
双击start.bat
输入书籍ID
